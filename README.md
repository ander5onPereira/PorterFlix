## 🚀 Como executar

- Clone o repositório
- Instale as dependências com `yarn`
- Altere o nome do arquivo `config.exemple.ts` para `config.ts` 
- Abra o arquivo `config.ts` adicione o `TOKEN` da api `themoviedb.org`
- Inicie seu app com `expo start`
